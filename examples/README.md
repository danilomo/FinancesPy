# FinancesPy Examples

This directory contains example scripts demonstrating FinancesPy features.

## predicate_demo.py

Demonstrates the predicate compiler functionality for filtering transactions based on complex expressions.

Run with:
```bash
python examples/predicate_demo.py
```

The demo shows:
1. Category-based filtering with exclusions
2. Value-based filtering
3. Date range filtering
4. Description pattern matching with regex
5. Complex combinations of multiple criteria
6. Boolean logic (AND, OR, NOT)
